# cats-discord
A simple Python script to monitor the journal output on your CATS Linux I-Gate / Digipeater and post the messages to a Discord channel via a webhook.

What is CATS? It's short for Communication And Telemetry System see here: https://cats.radio/

For info on creating Discord webhooks see here: https://support.discord.com/hc/en-us/articles/228383668-Intro-to-Webhooks
